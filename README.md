🏥 Smart Hospital Intelligence & Decision Support System

📌 Overview

This project is an AI-powered Hospital Intelligence Dashboard designed to help hospital administrators make faster and smarter decisions using healthcare data.

The system performs data cleaning, analytics, machine learning-based patient risk prediction, and decision support recommendations. Interactive Power BI dashboards provide insights into patient flow, disease trends, hospital performance, and operational efficiency.

🚩 Problem Statement

Hospitals generate a large amount of patient data every day. Analyzing this information manually is slow and difficult.

Hospital administrators need an intelligent system that can:

- Monitor hospital performance
- Identify high-risk patients
- Reduce waiting times
- Improve resource allocation
- Support faster decision making

💡 Proposed Solution

This project builds an AI-powered analytics platform that:

- Cleans raw hospital data
- Stores data in SQLite
- Predicts patient risk using Machine Learning
- Generates operational recommendations
- Visualizes insights in Power BI dashboards

- 🛠 Technologies Used

- Python
- Pandas
- NumPy
- SQLite
- Scikit-Learn
- Power BI
- VS Code

📈 Results

The dashboard helps hospital administrators:

- Identify high-risk patients quickly
- Monitor hospital performance
- Improve bed management
- Reduce patient waiting time
- Support data-driven decision making
